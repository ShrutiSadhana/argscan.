# ARGScan — Antibiotic Resistance Gene Detector

Detect antibiotic resistance genes (ARGs) in genomic sequences using k-mer indexing.

## Project structure

```
argscan/
├── arg_detector/
│   ├── __init__.py        Package entry point
│   ├── parser.py          FASTA parser — FastaRecord dataclasses
│   ├── kmer.py            k-mer generation and positional indexing
│   ├── database.py        Resistance gene database loader
│   ├── detector.py        Core detection logic (Jaccard k-mer scoring)
│   ├── cli.py             Command-line interface
│   ├── data/
│   │   ├── resistance_genes.fasta   Reference database
│   │   └── query.fasta              Example query
│   └── tests/
│       ├── test_kmer.py
│       ├── test_database.py
│       └── test_detector.py
├── argscan.html           Frontend (single-file, open in browser)
├── pyproject.toml         Package metadata and CLI entry point
├── run.bat                One-click Windows launcher
└── README.md
```

## Installation

```bash
cd argscan
pip install -e .
```

## Usage

### CLI

```bash
arg-detector \
  --query arg_detector/data/query.fasta \
  --database arg_detector/data/resistance_genes.fasta \
  --kmer-size 15 \
  --min-score 0.1
```

### Windows one-click

Double-click `run.bat`

### Web UI

Open `argscan.html` in any browser — no server required.

## How it works

1. **parser.py** reads FASTA files into `FastaRecord` dataclasses
2. **kmer.py** generates all k-mers and builds a positional index
3. **database.py** loads the reference resistance gene database
4. **detector.py** intersects query k-mers with reference k-mers and computes a Jaccard coverage score
5. **cli.py** wires everything together and formats the output

## Algorithm

```
score = |query_kmers ∩ ref_kmers| / |ref_kmers|
```

A score of 1.0 means complete coverage of the reference gene's k-mers in the query.

## Database format

FASTA headers must follow the pipe-delimited format:

```
>gene_id|gene_name|family
ATGCGATCG...
```

Replace `data/resistance_genes.fasta` with a download from
[CARD](https://card.mcmaster.ca/download) or [ResFinder](https://cge.food.dtu.dk/services/ResFinder/)
for production use.
